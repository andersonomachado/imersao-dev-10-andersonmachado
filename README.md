# Guia Interativo das Madeiras Brasileiras

Este projeto é uma aplicação web que reúne e disponibiliza as principais informações sobre as madeiras brasileiras catalogadas pelo Laboratório de Produtos Florestais (LPF) do Serviço Florestal Brasileiro (SFB).

## Objetivo

Facilitar a pesquisa, consulta e divulgação de dados oficiais sobre espécies de madeiras nativas e exóticas do Brasil, oferecendo uma interface intuitiva e acessível para estudantes, profissionais do setor florestal, pesquisadores e interessados em biodiversidade.

## Funcionalidades

*   **Busca inteligente:** Pesquise por nome científico, nome comercial ou características das madeiras.
*   **Base de dados completa:** Informações técnicas, origem, distribuição geográfica, propriedades físicas, usos comerciais e links para fichas técnicas oficiais do LPF.
*   **Interface amigável:** Navegação simples, visual limpo e recursos interativos.
*   **Links diretos:** Acesse rapidamente as páginas oficiais de cada espécie no site do LPF.
*   **Acesso universal:** Compatível com computadores, tablets e smartphones.

## Diferenciais

*   Dados atualizados conforme o banco oficial do LPF/SFB.
*   Ferramenta de apoio à pesquisa, ensino e tomada de decisão.
*   Código aberto e fácil de adaptar para novas funcionalidades ou integrações (ex: API CKAN).

## Como usar

1.  Acesse o site: <https://andersonomachado.github.io/imersao-dev-10-andersonmachado/>
2.  Utilize o campo de busca para encontrar informações sobre a madeira desejada.
3.  Explore os detalhes técnicos e acesse os links oficiais para aprofundar sua pesquisa.
